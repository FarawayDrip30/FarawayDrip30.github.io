Birdpals

Birdpals was a club penguin-esque game that me and some friends were making for quite some time, but unforunately when the colour party was launched, whenever one specific guy joined the game, the server crashed, and I couldn't fix it and so gave up. This is a singleplayer port of the game. The project lead guy is currently making it into another project called "Mythkins", but I'm not working on that.
