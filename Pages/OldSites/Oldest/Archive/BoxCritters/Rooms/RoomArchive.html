How To Use

Copy the Code Below (Scroll Down or Press the Button to go there) then Paste the Code into the Console in Box Critters.

You can Usually Open your Browser's Console by Pressing F12 or Going to More Tools - Developer Tools

Once you have Done that, type "timeTravel(year,event,part)" into the Console, but Replace year, event, and part with info from the List Below, Making Sure they are all in Quotation Marks.

For Example, to go to the Second Part of the Halloween 2021 Event you would type: timeTravel("2021","Halloween","Part2")
